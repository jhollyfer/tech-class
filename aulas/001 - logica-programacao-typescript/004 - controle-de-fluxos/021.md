Arrays - Armazenando vários dados em listas
Olá, pessoal! Sou o André Noel e nesta aula de Lógica de Programação, exploramos como armazenar múltiplos dados em uma única variável usando arrays, ou listas, em Python. Aprendemos a criar, acessar e modificar elementos, além de misturar tipos de dados. Também discutimos como percorrer listas para calcular a média de notas, utilizando loops. Essa base é fundamental para avançarmos em projetos mais complexos. Fiquem ligados para a próxima aula, onde aplicaremos esses conceitos em um joguinho!

🧑‍💻 Código desenvolvido em aula:

notas = [9, 9.5, 10, 9.8]

media = 0
for nota in notas:
media += nota

media /= 4

print(f'A média é {media}')
