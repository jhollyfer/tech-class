# Aula 02 — Fonte, Parágrafo & Estilos de Texto

**Curso:** Qualificação Profissional em Informática Avançada  
**Componente:** Word — Editor de Texto (40h)  
**Carga horária:** 4 horas  
**Vídeos de referência:** #06 e #07 — Curso em Vídeo

---

## Objetivo

Formatar textos com precisão usando fonte e parágrafo, e padronizar documentos inteiros automaticamente com estilos de texto.

---

## 1. Formatação de Fonte

Tudo fica em **Página Inicial → grupo Fonte**.  
Para abrir todas as opções de uma vez: `Ctrl + D`.

### Tipos de fonte

| Tipo                    | Exemplos                  | Quando usar                           |
| ----------------------- | ------------------------- | ------------------------------------- |
| Serif (com serifa)      | Times New Roman, Garamond | Documentos acadêmicos e formais       |
| Sans-serif (sem serifa) | Arial, Calibri            | Documentos modernos e digitais        |
| Monoespaçada            | Courier New               | Código de programação e texto técnico |

> **Regra:** Use no máximo 2 famílias de fonte por documento.

### Atalhos de caractere

| Atalho          | Efeito            | Quando usar                       |
| --------------- | ----------------- | --------------------------------- |
| `Ctrl + N`      | **Negrito**       | Termos importantes, destaques     |
| `Ctrl + I`      | _Itálico_         | Títulos de obras, estrangeirismos |
| `Ctrl + S`      | Sublinhado        | Usar com moderação                |
| `Ctrl + D`      | Janela Fonte      | Todas as opções avançadas         |
| `Ctrl + Espaço` | Limpar formatação | Volta ao estilo Normal            |
| `Ctrl + ]`      | Aumentar fonte    | Sem usar o mouse                  |
| `Ctrl + [`      | Diminuir fonte    | Sem usar o mouse                  |

### Pincel de Formatação

- Copia a formatação de um trecho e aplica em outro
- Ícone de pincel no grupo **Área de Transferência**
- **1 clique** = aplica em um trecho
- **Duplo clique** = modo contínuo (vários trechos)

## 2. Formatação de Parágrafo

Acesse em **Página Inicial → grupo Parágrafo** ou botão direito → Parágrafo.

### O erro mais comum

❌ **Errado:** pressionar Enter duas vezes entre parágrafos  
✅ **Certo:** configurar o campo **"Espaçamento Depois"** em Parágrafo

> **Nunca use Enter duplo para espaçar parágrafos.** Use o campo "Espaçamento Depois" — isso garante consistência e permite que o sumário automático funcione.

### Alinhamento

| Atalho     | Tipo         | Quando usar                     |
| ---------- | ------------ | ------------------------------- |
| `Ctrl + Q` | Esquerda     | Textos informais, e-mails       |
| `Ctrl + E` | Centralizado | Títulos, capas                  |
| `Ctrl + G` | Direita      | Datas, assinaturas              |
| `Ctrl + J` | Justificado  | Padrão ABNT para corpo do texto |

### Espaçamento entre linhas

| Atalho     | Espaçamento   | Uso                                 |
| ---------- | ------------- | ----------------------------------- |
| `Ctrl + 1` | Simples (1,0) | Rodapés, legendas                   |
| `Ctrl + 5` | 1,5 linhas    | **Padrão ABNT** para corpo do texto |
| `Ctrl + 2` | Duplo (2,0)   | Documentos para revisão             |

### Configuração ABNT completa — passo a passo

1. Fonte: **Arial** ou **Times New Roman**, tamanho **12**
2. Alinhamento: **Justificado** — `Ctrl + J`
3. Espaçamento entre linhas: **1,5** — `Ctrl + 5`
4. Espaçamento Depois: **0pt** — Parágrafo → Espaçamento → Depois: 0
5. Recuo de primeira linha: **1,25cm** — Parágrafo → Recuos Especiais → Primeira linha

## 3. Estilos de Texto

Um **estilo** é um conjunto de formatações com um nome. Ao modificar o estilo, todos os trechos com ele atualizam automaticamente de uma vez.

### Hierarquia de estilos

| Estilo       | Uso                               |
| ------------ | --------------------------------- |
| **Título 1** | Capítulos principais              |
| **Título 2** | Subcapítulos                      |
| **Título 3** | Subsubcapítulos                   |
| **Normal**   | Corpo do texto                    |
| **Citação**  | Citações longas (ABNT: recuo 4cm) |

> **Fundamental:** Use sempre Título 1, 2 e 3 para estruturar o documento. O **sumário automático** (Aula 06) só funciona com eles.

### Por que usar estilos

| Sem estilos                                          | Com estilos                                 |
| ---------------------------------------------------- | ------------------------------------------- |
| 80 títulos para mudar de cor: 30 minutos de trabalho | Modificar o estilo uma vez: 10 segundos     |
| Inconsistências visuais em documentos longos         | Padronização automática em todo o documento |
| Sumário automático não funciona                      | Sumário gerado em 1 clique                  |

### Como modificar um estilo

1. Botão direito no estilo na galeria → **Modificar**
2. Altere: fonte, tamanho, cor, espaçamento
3. Clique em **OK** — todos os trechos com aquele estilo atualizam

### Como criar um estilo novo

1. Formate um parágrafo manualmente do jeito desejado
2. Clique na caixa de estilos (mostra o nome do estilo atual)
3. Digite o nome do novo estilo e pressione Enter

## 4. Checklist da Aula

- [ ] Configurei um parágrafo completo com as normas ABNT
- [ ] Testei negrito (`Ctrl+N`), itálico (`Ctrl+I`) e limpeza de formatação (`Ctrl+Espaço`)
- [ ] Usei o Pincel de Formatação para copiar aparência entre trechos
- [ ] Apliquei os estilos Título 1, Título 2 e Normal em um documento
- [ ] Modifiquei um estilo e vi todos os títulos atualizarem automaticamente
