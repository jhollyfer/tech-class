# Aula 05 — Modelagem de Negócios

**Disciplina:** Empreendedorismo e Inovação na Saúde | **Duração:** 2 horas

---

## 🎯 Objetivos da Aula

- Compreender o conceito de modelo de negócios
- Conhecer a estrutura básica de um negócio na área da saúde
- Identificar público-alvo e construir uma proposta de valor
- Analisar exemplos práticos de modelos de negócio na saúde

---

## 📖 Conteúdo

### 1. O Que é um Modelo de Negócio?

Um modelo de negócio descreve **como uma empresa cria, entrega e captura valor**. Simplificando:

> Como vou ajudar as pessoas? → **Como vou entregar essa ajuda?** → Como vou ganhar dinheiro com isso?

---

### 2. O Business Model Canvas (Canvas Simplificado)

O Canvas é uma ferramenta visual que organiza as principais partes de um negócio em **9 blocos**:

| Bloco                     | Pergunta Central                              |
| ------------------------- | --------------------------------------------- |
| **Proposta de Valor**     | Que problema resolvo? Que benefício entrego?  |
| **Segmentos de Clientes** | Para quem? Quem é meu público?                |
| **Canais**                | Como chego até meus clientes?                 |
| **Relacionamento**        | Como me relaciono com eles?                   |
| **Fontes de Receita**     | Como vou ganhar dinheiro?                     |
| **Recursos Principais**   | O que preciso para operar?                    |
| **Atividades-Chave**      | O que faço todos os dias para entregar valor? |
| **Parcerias**             | Com quem preciso me aliar?                    |
| **Estrutura de Custos**   | Quais são meus principais gastos?             |

---

### 3. Proposta de Valor na Saúde

A proposta de valor responde: **"Por que o cliente/paciente vai escolher você?"**

#### Fórmula da Proposta de Valor:

> Para **[quem]** que **[problema/necessidade]**, meu serviço **[solução]** que **[benefício diferenciado]**, diferente de **[alternativas existentes]**.

**Exemplo:**

> Para **idosos em Tabatinga** que **precisam de cuidados domiciliares**, meu serviço de **home care** oferece **atendimento humanizado e personalizado na própria casa**, diferente de **ter que se deslocar até uma UBS ou hospital**.

---

### 4. Exemplos Práticos de Modelos de Negócio na Saúde

#### Home Care

| Componente        | Descrição                                              |
| ----------------- | ------------------------------------------------------ |
| Proposta de Valor | Cuidado de saúde no conforto do lar                    |
| Público-Alvo      | Idosos, pós-operatório, portadores de doenças crônicas |
| Canal             | Indicação médica, redes sociais, WhatsApp              |
| Receita           | Diária ou pacote mensal de visitas                     |
| Parceiro Chave    | Médicos que indicam, planos de saúde                   |

#### Consultoria em Saúde Ocupacional

| Componente        | Descrição                                               |
| ----------------- | ------------------------------------------------------- |
| Proposta de Valor | Reduzir afastamentos e custos com acidentes de trabalho |
| Público-Alvo      | Empresas com mais de 20 funcionários                    |
| Canal             | Visitas comerciais, LinkedIn                            |
| Receita           | Contrato mensal de assessoria                           |
| Parceiro Chave    | Médico do trabalho, SESMT                               |

#### Farmácia Clínica

| Componente        | Descrição                                                 |
| ----------------- | --------------------------------------------------------- |
| Proposta de Valor | Uso seguro e eficaz dos medicamentos                      |
| Público-Alvo      | Portadores de doenças crônicas polimedicados              |
| Canal             | Própria farmácia, indicação médica                        |
| Receita           | Consulta farmacêutica, serviço de revisão de medicamentos |
| Parceiro Chave    | Médicos, planos de saúde                                  |

---

### 5. Identificando Seu Público-Alvo (Persona)

Criar uma **persona** significa imaginar quem é o seu cliente ideal:

**Exemplo de Persona:**

```
Nome: Dona Maria
Idade: 68 anos
Situação: Diabética, hipertensa, mora sozinha
Problema: Dificuldade de se locomover até UBS
Necessidade: Controle glicêmico e de pressão em casa
O que ela valoriza: Atenção, confiança, pontualidade
```

## ✍️ Atividade Prática

### Construção do Mini-Canvas

**Tempo:** 35 minutos | **Grupo de 3 a 4 pessoas**

Usando a folha do Canvas (fornecida pelo professor), preencha os **9 blocos** para a ideia de negócio que o grupo identificou na aula anterior.

**Dicas:**

- Comece pela Proposta de Valor e pelo Público-Alvo
- Seja específico: quem é exatamente seu cliente?
- Pense em como vai ganhar dinheiro de forma sustentável
- Apresente o Canvas para a turma em 3 minutos

## 🔑 Pontos-Chave para Fixar

1. Modelo de negócio = Como criar, entregar e capturar valor
2. O Canvas organiza um negócio em 9 blocos interligados
3. Comece sempre pela **Proposta de Valor** — sem ela, nada faz sentido
4. Conhecer sua persona (cliente ideal) é fundamental para tomar decisões certas
